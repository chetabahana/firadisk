# firadisk
A Windows driver to read GRUB4DOS' drive map table and use GRUB4DOS RAM drives in Windows. 

Can run in
- Windows XP 32-bit
- Windows Server 2003 32-bit
- Windows 7 32-bit
- Windows 7 64-bit (Test Mode)
